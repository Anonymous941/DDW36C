# DDW36C

This is the dump of my DDW36C router, for [creating a device profile in bcm-utils](https://github.com/jclehner/bcm2-utils?tab=readme-ov-file#writing-a-device-profile).

* `bootloader.bin`: dump of the bootloader
* `image1.bin`, `image2.bin`: dump of these firmware regions
* `flash_show.txt`: output of `/flash/show` command
* `flash_help_open.txt`: output of `/flash/help open` command
* `version.txt`: output of `/version` command
